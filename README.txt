
It is a maven project so any one can import any populer java IDE. It is developed using eclipse (Version: Neon Release (4.6.0)) IDE.

This Application developed using spring version 4 and java 8. To expose rest web service spring controller has been used.

There is two rest service inside it.
1. /rest/atms : To get all ATM data.
2. /rest/atms/{city} : To get ATM data for givn city.

To test rest web service have writen integration test cases.

To serch and view data have created one html page (resources/atms.html).
